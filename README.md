# SchraubenSchütte - Lagerverwaltung für Kleinteile

Eine professionelle WPF-Anwendung zur Verwaltung von Lagerbeständen, speziell entwickelt für die Organisation von Schrauben und Kleinteilen. Das Projekt demonstriert den Einsatz von **C#**, **WPF (MVVM)** und **SQLite**-Datenbanken.

## Features
* **Bestandsaufnahme:** Hinzufügen von neuen Artikeln mit Details (Material, Maße, Preise).
* **Datenhaltung:** Permanente Speicherung in einer lokalen SQLite-Datenbank.
* **Übersicht:** Dynamische Anzeige aller Bestände in einer tabellarischen Ansicht.
* **Sicherheit:** Validierung von Benutzereingaben zur Vermeidung von Fehlern.
* **Bildverwaltung:** Verknüpfung von Produktbildern zu den jeweiligen Datensätzen.
* **Exportieren:** Tabelle als Excel abspeichern.
